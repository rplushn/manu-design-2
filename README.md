# MANU Design 2

Diseño minimalista para la landing page de MANU.

## Características del diseño

- Border-radius: 0 (esquinas completamente rectas)
- Tipografía delgada y pequeña
- Paleta monocromática (blanco, negro, grises)
- Espaciado generoso y limpio
- 12 secciones optimizadas

## Instalación

```bash
npm install
# o
pnpm install
# o
yarn install
```

## Desarrollo

```bash
npm run dev
# o
pnpm dev
# o
yarn dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Deploy en Vercel

1. Conecta este repositorio con tu cuenta de Vercel
2. Vercel detectará automáticamente que es un proyecto Next.js
3. Haz click en Deploy

¡Listo!
